## Infrared Proximity Sensor
![Infrared Proximity Sensor](https://abacasstorageaccnt.blob.core.windows.net/cirkit/7c39c73f-23f9-44c8-81a3-428e26415a55.png "Infrared Proximity Sensor")

> เซ็นเซอร์ตรวจจับวัตถุด้วยอินฟราเรดเป็นอุปกรณ์ที่ตรวจจับการมีอยู่ของวัตถุที่อยู่ใกล้เคียงโดยการปล่อยแสงอินฟราเรด (IR) และวัดสัญญาณสะท้อนกลับ มีการใช้งานอย่างแพร่หลายในแอปพลิเคชันที่ต้องการการตรวจจับวัตถุแบบไม่สัมผัส เซ็นเซอร์นี้มักพบในระบบอัตโนมัติ หุ่นยนต์ การตรวจจับสิ่งกีดขวาง และอินเทอร์เฟซแบบไร้สัมผัส

# การใช้งานทั่วไป:
- การตรวจจับสิ่งกีดขวางในหุ่นยนต์
- ประตูอัตโนมัติและเจลล้างมือ
- หุ่นยนต์ติดตามเส้น
- ระบบไฟส่องสว่างตามระยะใกล้
- ระบบอัตโนมัติทางอุตสาหกรรมสำหรับการนับหรือคัดแยกวัตถุ


# ข้อกำหนดทางเทคนิค
 ด้านล่างนี้คือรายละเอียดทางเทคนิคที่สำคัญของเซ็นเซอร์ตรวจจับระยะใกล้แบบอินฟราเรดทั่วไป:
 
พารามิเตอร์	ค่า
แรงดันไฟฟ้าใช้งาน	3.3V ถึง 5V
กระแสไฟฟ้าขณะใช้งาน	20mA ถึง 50mA
ระยะการตรวจจับ	2 ซม. ถึง 30 ซม. (แตกต่างกันไปตามรุ่น)
ประเภทเอาต์พุต	ดิจิทัล (สูง/ต่ำ) หรือ อนาล็อก
ความยาวคลื่นของแสงอินฟราเรด	~940 นาโนเมตร
เวลาตอบสนอง	< 2 มิลลิวินาที
อุณหภูมิในการทำงาน	-10°C ถึง 50°C

## การกำหนดค่าและคำอธิบายของพิน

เซ็นเซอร์ตรวจจับระยะใกล้แบบอินฟราเรดโดยทั่วไปจะมีขาเชื่อมต่อสามขาขึ้นไป ตารางด้านล่างนี้แสดงรายละเอียดการจัดเรียงขาเชื่อมต่อที่พบได้ทั่วไป:

ชื่อพิน	คำอธิบาย
วีซีซี	ขาจ่ายไฟ (3.3V หรือ 5V)
ก.น.	การต่อสายดิน
ออก	ขาเอาต์พุต (สัญญาณดิจิทัลหรืออนาล็อก)
EN (ไม่บังคับ)	เสียบพินเพื่อเปิดหรือปิดใช้งานเซ็นเซอร์
